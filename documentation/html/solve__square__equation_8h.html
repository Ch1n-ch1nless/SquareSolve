Solve linear equation.

Parameters
[in]UsersTrinomialstructure in which coefficient values are found
[in]Rootstructure with root values
[out]EquationRootsQuantitynumber of roots

The function solves a linear equation. If the coefficient before x is not 0, then return ONE_ROOT. value of root = -free term / coefficient. If the coefficient before x is 0 and free term is 0, then return INFINITE_ROOTS_QUANTITY. In other cases return NO_ROOTS.

Solve square equation.

Parameters
[in]UsersTrinomial- structure in which coefficient values are found
[out]Root- structure with root values
Returns
EquationRootsQuantity number of roots

Algorithm: If the senior coefficient is 0, then starts SolveLinearEquation(). Else calculate value of Discriminant. If Discriminant < 0 return NO_ROOTS. If Discriminant = 0 calculate calculates the root value by formula and return ONE_ROOT. Else calculates the values of roots by formula and return TWO_ROOTS.
